# Slider Mute

This is a simple mod that adds <cr>Mute</c> buttons next to the **Music** and **SFX** sliders in the Pause Menu.

## Features
### Auto Mute Music
- When entering a level it will automaticly <cr>Mute</c> the **Music**

### Auto Mute SFX
- When entering a level it will automaticly <cr>Mute</c> the **SFX**

### Use Alternate Textures
- this just swaps out the normal `GJ_audioBtnOn_001.png` with `GJ_fxBtnOn_001.png`
