# Slider Mute

This is a simple quality-of-life mod that adds convenient <cr>Mute</c> buttons directly next to the **Music** and **SFX** sliders in the Pause Menu.

## Features

### Auto-Mute Music
- Automatically applies <cr>Mute</c> to the **Music** whenever you enter a level.

### Auto-Mute SFX
- Automatically applies <cr>Mute</c> to the **SFX** whenever you enter a level.

### Alternate Textures
- Swaps out the default `GJ_audioBtnOn_001.png` texture with `GJ_fxBtnOn_001.png` for a customized look.
